Why does this repo exist?
Honestly, if I don't write it down, I forget it. This space helps me:

Actually remember the syntax for that one specific command.

Understand why a tool works, rather than just blindly copying and pasting from tutorials.

Have a tangible record of my progress for the days when imposter syndrome hits hard.

Maybe help out someone else who is currently stuck exactly where I was yesterday.

Brain Dumps – Thoughts on the mindset required for this field, handling frustration, and learning how to think like an attacker.
